# 📦 pco「pip current origin」

## 安装

```bash
pip install pipco
```

## 查看帮助

```bash
pco help
```

## 镜像源使用方法

在 `pip install` 安装包名称 `-i` 镜像源

```bash
-i http://pypi.douban.com/simple/ --trusted-host=pypi.douban.com/simple
```
